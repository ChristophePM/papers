MJEPA: A Simple and Scalable Joint-Embedding Predictive Architecture for Audio-Visual Learning

Revant Teotia, Adrien Bardes, Michael Rabbat, Sumit Chopra, Matthew Muckley, Nicolas Ballas — FAIR at Meta & NYU

Computer Science Self-Supervised Learning Audio-Visual 2026 Technical
Source
arXiv:2606.25225v1 · HTML version
TL;DR

MJEPA extends Joint-Embedding Predictive Architectures to audio-visual learning using a single unified encoder for both modalities. The critical insight: a shared encoder hurts without cross-modal prediction, but with it each modality boosts the other. A frozen ViT-g beats prior frozen baselines by over +6.8 mAP on AudioSet-20K — trained entirely from scratch, no ImageNet pretraining.

Key Takeaways

Shared encoders fail without alignment. A naive shared encoder degraded both modalities below unimodal baselines, contradicting the intuition that parameter sharing automatically helps. → Ablation
Cross-modal prediction is the critical ingredient. Adding lightweight MLP-based alignment flipped the architecture from harmful to beneficial — audio mAP jumped from 28.70 to 33.52. → Architecture
Frozen representations beat finetuned baselines. On ESC-50 (96.9%) and FSD50K (65.8 mAP), frozen MJEPA surpasses even finetuned prior methods like EquiAV. → Results
Scales cleanly to 1B parameters. The framework handles heterogeneous data (mixed unimodal + multimodal) and scales to ViT-g with simultaneous gains across all benchmarks. → Ablation
Simplicity wins. No contrastive losses, no reconstruction objectives, no heavy augmentations, no pretrained initialization — just prediction. → vs. Prior Work

Core Architecture

Unlike prior audio-visual self-supervised methods that use separate modality-specific encoders with complex loss combinations, MJEPA employs deliberate simplicity with three components:

1. Unified Encoder

A single Vision Transformer processes audio, video, or combined audio-video inputs through shared weights. Only the tokenization layers (patch embedding) and positional embeddings are modality-specific — everything else is shared.

2. Intra-Modal Prediction

Standard JEPA masking applied independently within each modality: predict masked token representations from visible context using L1 distance loss. Multi-level features from several encoder layers are fused for richer hierarchical representations.

3. Cross-Modal Prediction

Lightweight MLPs predict pooled last-layer features across modalities (audio→video, video→audio, audio↔audio-video). Global pooling is used because there is no natural token-level correspondence between audio and video. This alignment objective is what transforms the shared encoder from harmful to beneficial — the paper's central contribution.

Progressive Ablation

The paper builds the final architecture incrementally, revealing that each component is essential. All numbers are audio / video mAP on AudioSet-20K with frozen evaluation:

  • Unimodal baselines (separate models): 30.87 / 19.84
  • Naive shared encoder (no alignment): 28.70 / 17.90 — both modalities degraded
  • + Cross-modal alignment: 33.52 / — — audio recovers and exceeds baseline
  • + Joint audio-video encoding: 38.89 / 25.38
  • + Data & model scaling (ViT-g, 1B params): 40.97 / 29.82
  • Audio-video combined: 45.44 mAP

The naive-to-aligned jump is the paper's most striking result: simply sharing parameters hurts, but adding cross-modal prediction makes both modalities stronger than they were alone.

Main Results

Audio-Visual (AudioSet-20K)

Frozen evaluation SOTA at 45.44 mAP (audio-video combined), outperforming previous best (EquiAV) by 4.3–6.8 mAP across modalities. Notably, the frozen video-only result (29.63) surpasses all reported finetuned baselines.

Audio-Only Tasks

Sets frozen representation SOTA across ESC-50 (96.9% accuracy) and FSD50K (65.8 mAP). Both numbers exceed finetuned EquiAV results, demonstrating strong positive transfer from video data improving audio representations.

Video-Only Tasks

Competitive with VJEPA2 despite using 10× less video data: Kinetics-400 at 85.0% (vs 86.6%) and Something-Something-v2 at 73.9% (vs 75.3%). Audio-video training provides a complementary learning signal that benefits video representations.

What Makes This Different

Previous audio-visual self-supervised methods relied on:

  • Separate modality-specific encoders
  • Combinations of contrastive losses and reconstruction objectives
  • Heavy augmentation pipelines
  • ImageNet pretraining as initialization

MJEPA uses none of these — a single shared encoder, one prediction-based objective, minimal augmentation, and trains from scratch. The elimination of these complexities, combined with explicit cross-modal alignment, achieves superior results across the board.

Conclusion

MJEPA demonstrates that a single, shared-weight Vision Transformer can learn strong audio-visual representations through prediction alone — no contrastive losses, reconstruction, or pretrained weights needed. The key enabler is cross-modal prediction: without it, shared encoders underperform; with it, modalities synergize. The approach scales smoothly to 1B parameters and sets new SOTA on frozen evaluation across audio, video, and audio-visual benchmarks.

So What?

This is a proof-of-concept for truly modality-agnostic architectures. If one encoder with one objective can handle audio + video better than specialized pipelines, the same pattern could extend to medical imaging (CT + MRI), robotics (vision + proprioception), or any domain where multiple data streams naturally co-occur. The simplicity makes it practical to scale and adapt — fewer moving parts means fewer things to tune.
